# VsixDL

Adds download button for VSCode extensions on the Visual Studio Marketplace.

## Features

- Adds "Download VSIX" button to extension pages
- Downloads extensions as .vsix files
- Works directly from marketplace website
- No need to use command line tools

## How It Works

Adds a download button next to the install button that downloads the extension package:

Perfect for offline installations or manual extension management.

## Installation

1. Install a userscript manager (Tampermonkey, Greasemonkey, etc.)
2. Install VsixDL
3. Visit any VSCode extension on marketplace.visualstudio.com
